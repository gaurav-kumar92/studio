# **App Name**: JS Playground

## Core Features:

- Code Editor: A Monaco-based code editor to write Javascript code.
- REPL Environment: An embedded, client-side Javascript runtime environment. Display output from the editor's code. Provide any error or debugging feedback to the user in the UI.
- Generative example tool: Provide an LLM-powered 'tool' button, allowing the LLM to write code for the user if it considers it appropriate.

## Style Guidelines:

- Primary color: Teal (#008080) to evoke trust and intelligence.
- Background color: Light gray (#E0E0E0) to reduce eye strain during prolonged use.
- Accent color: Lime green (#32CD32) to highlight output, actions, and interactivity.
- Body and headline font: 'Inter' sans-serif font to provide a modern, readable UI. Suitable for UI elements and longer text.
- Code font: 'Source Code Pro' monospaced font for Javascript code and error output.
- Split screen layout, featuring editor on the left, and REPL environment on the right.
